bool parcNetwork_Inet4Equals ( const struct sockaddr_in *  x,
const struct sockaddr_in *  y 
)

Return true if two sockaddr_in instances are equal.

The equality function that this evaluates must implement the following equivalence relations on non-null instances:

  • It is reflexive: for any non-null reference value x, parcNetwork_Inet4Equals(x, x) must return true.
  • It is symmetric: for any non-null reference values x and y, parcNetwork_Inet4Equals(x, y) must return true if and only if parcNetwork_Inet4Equals(y x) returns true.
  • It is transitive: for any non-null reference values x, y, and z, if parcNetwork_Inet4Equals(x, y) returns true and parcNetwork_Inet4Equals(y, z) returns true, then parcNetwork_Inet4Equals(x, z) must return true.
  • It is consistent: for any non-null reference values x and y, multiple invocations of parcNetwork_Inet4Equals(x, y) consistently return true or consistently return false.
  • For any non-null reference value x, equals(x, NULL)) must return false.
Parameters
[in]xA pointer to a struct sockaddr_in instance.
[in]yA pointer to a struct sockaddr_in instance.s
Returns
true the given sockaddr_in instances are equal.
false the given sockaddr_in instances are not equal.

bool parcNetwork_IsSocketLocal ( struct sockaddr *  sock)

Determines if a socket is local.

A socket is local if: (a) PF_LOCAL/AF_UNIX, or (b) PF_INET and on the 127.0.0.0/8 network, or (c) PF_INET6 and equal to the ::1 address. Anything else is considered non-local.

Parameters
[in]sockA socket structure
Returns
true The socket represents a local/loopback address
false The socket is not a local/loopback address

Example:

1 {
2  struct sockaddr *s = parcNetwork_SockAddress("127.1.1.1", 5900);
3  assertTrue(parcNetwork_IsSocketLocal(s), "This will not assert");
4  parcMemory_Deallocate((void **)&s);
5 }

The standard (IEEE 802) format for printing MAC-48 addresses in human-friendly form is six groups of two hexadecimal digits, separated by hyphens (-) or colons (:), in transmission order (e.g.

01-23-45-67-89-ab or 01:23:45:67:89:ab). This form is also commonly used for EUI-64. Another convention used by networking equipment uses three groups of four hexadecimal digits separated by dots (.) (e.g. 0123.4567.89ab ), again in transmission order.

Parse a link address, expressed as a nul-terminated C string of the form link:// followed by hexadecimal numbers representing single byte values, each value separated by either a ':' or '-' character.

The standard (IEEE 802) format for printing MAC-48 addresses in human-friendly form is six groups of two hexadecimal digits, separated by hyphens (-) or colons (:), in transmission order (e.g. 01-23-45-67-89-ab or 01:23:45:67:89:ab). This form is also commonly used for EUI-64.

Another convention used by networking equipment uses three groups of four hexadecimal digits separated by dots (.) (e.g. 0123.4567.89ab ), again in transmission order.

struct sockaddr* parcNetwork_SockAddress ( const char *  address,
in_port_t  port 
)

Parses any string in to an address, if possible.

The string may be an IPv6, IPv6 or hostname. If the string does not match an IPv4 or IPv6 nominal format, it will try to resolve the string as a hostname. If that fails, the function will return NULL.

IMPORTANT: the returned pointer is allocated with parcMemory_Allocate() and you must use parcMemory_Deallocate().

Parameters
[in]addressAn IPv4, IPv6, or hostname
[in]portthe port address
Returns
NULL Could not parse the address or resolve as a hostname
non-NULL a valid sockaddr. You shoule examine sockaddr->sa_family to determine IPv4 or IPv6.

Example:

1 {
2  struct sockaddr *addr;
3  addr = parcNetwork_SockAddress("1.2.3.4", 555);
4  assertTrue(addr && addr->sa_family == AF_INET, "Addr not IPv4 for a dotted quad.");
5  struct sockaddr_in *addr_in = (struct sockaddr_in *) addr;
6  // ...
7  parcMemory_Deallocate((void **)&addr);
8 
9  addr = parcNetwork_SockAddress("fe80::aa20:66ff:fe00:314a", 555);
10  assertTrue(addr && addr->sa_family == AF_INET6, "Addr not IPv6.");
11  struct sockaddr_in6 *addr_in6 = (struct sockaddr_in6 *) addr;
12  // ...
13  parcMemory_Deallocate((void **)&addr);

15  addr = parcNetwork_SockAddress("alpha.parc.com", 555);

17  // ...
