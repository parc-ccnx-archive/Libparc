Detailed Description

Queue buffer events.

Provides a facade implementing many regularly available event functions. This is an interface that software implementors may use to substitute different kinds of underlying implementations of these event management functions. Notable examples are libevent and libev.

PARCEventQueue_Callback *  readCallback,
PARCEventQueue_Callback *  writeCallback,
PARCEventQueue_EventCallback *  eventCallback,
void *  user_data 
)

Set callbacks on a buffer event instance.

The event instance is passed in. You can disable a callback by passing NULL instead of the callback function. NB: all the callback functions on a bufferevent share a single user_data value, so changing user_data will affect all of them.

Parameters
[in]eventInstance- event instance
[in]readCallback- callback for read events
[in]writeCallback- callback for write events
[in]eventCallback- callback for non-read/write events
[in]user_data- data passed along in callback arguments

Example:

1 void Callback(PARCEventType type, void *user_data)
2 {
3  printf("Received event of type=%d\n", type);
4 }
5 ...
6 {
7 ...
8  parcEventQueue_SetCallbacks(eventInstance, Callback, NULL, NULL, user_data);
9  parcEventQueue_Enable(eventInstance, PARCEventType_Read);
10 }
