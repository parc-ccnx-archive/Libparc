PARCInMemoryVerifier* parcInMemoryVerifier_Acquire ( const PARCInMemoryVerifier *  verifier)

Increase the number of references to the given PARCInMemoryVerifier instance.

A new instance is not created, only that the given instance's reference count is incremented. Discard the acquired reference by invoking parcInMemoryVerifier_Release().

Parameters
[in]signerA pointer to a PARCInMemoryVerifier instance.
Returns
NULL An error occurred.
non-NULL A pointer to a PARCInMemoryVerifier instance.

Example:

1 {
2  PARCInMemoryVerifier *verifier = parcInMemoryVerifier_Create();
3  PARCInMemoryVerifier *handle = parcInMemoryVerifier_Acquire(signer);
4  // use the handle instance as needed
5 }

void parcInMemoryVerifier_Release ( PARCInMemoryVerifier **  verifierPtr)

Release a previously acquired reference to the specified instance, decrementing the reference count for the instance.

The pointer to the instance is set to NULL as a side-effect of this function.

If the invocation causes the last reference to the instance to be released, the instance is deallocated and the instance's implementation will perform additional cleanup and release other privately held references.

The contents of the dealloced memory used for the PARC object are undefined. Do not reference the object after the last release.

Parameters
[in,out]verifierPtrA pointer to a pointer to the instance to release.

Example:

1 {
2  PARCInMemoryVerifier *verifier = parcInMemoryVerifier_Create();
3 
4  parcInMemoryVerifier_Release(&verifier);
5 }
