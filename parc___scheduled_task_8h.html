bool parcScheduledTask_Equals ( const PARCScheduledTask *  x,
const PARCScheduledTask *  y 
)

Determine if two PARCScheduledTask instances are equal.

The following equivalence relations on non-null PARCScheduledTask instances are maintained: *

  • It is reflexive: for any non-null reference value x, parcScheduledTask_Equals(x, x) must return true.
  • It is symmetric: for any non-null reference values x and y, parcScheduledTask_Equals(x, y) must return true if and only if parcScheduledTask_Equals(y x) returns true.
  • It is transitive: for any non-null reference values x, y, and z, if parcScheduledTask_Equals(x, y) returns true and parcScheduledTask_Equals(y, z) returns true, then parcScheduledTask_Equals(x, z) must return true.
  • It is consistent: for any non-null reference values x and y, multiple invocations of parcScheduledTask_Equals(x, y) consistently return true or consistently return false.
  • For any non-null reference value x, parcScheduledTask_Equals(x, NULL) must return false.
Parameters
[in]xA pointer to a valid PARCScheduledTask instance.
[in]yA pointer to a valid PARCScheduledTask instance.
Returns
true The instances x and y are equal.

Example:

1 {
2  PARCScheduledTask *a = parcScheduledTask_Create();
3  PARCScheduledTask *b = parcScheduledTask_Create();
4 
5  if (parcScheduledTask_Equals(a, b)) {
6  printf("Instances are equal.\n");
7  }
8 
9  parcScheduledTask_Release(&a);
10  parcScheduledTask_Release(&b);

PARCHashCode parcScheduledTask_HashCode ( const PARCScheduledTask *  instance)

Returns a hash code value for the given instance.

The general contract of HashCode is:

Whenever it is invoked on the same instance more than once during an execution of an application, the HashCode function must consistently return the same value, provided no information used in a corresponding comparisons on the instance is modified.

This value need not remain consistent from one execution of an application to another execution of the same application. If two instances are equal according to the parcScheduledTask_Equals method, then calling the parcScheduledTask_HashCode method on each of the two instances must produce the same integer result.

It is not required that if two instances are unequal according to the parcScheduledTask_Equals function, then calling the parcScheduledTask_HashCode method on each of the two objects must produce distinct integer results.

Parameters
[in]instanceA pointer to a valid PARCScheduledTask instance.
Returns
The hashcode for the given instance.

void parcScheduledTask_Release ( PARCScheduledTask **  instancePtr)

Release a previously acquired reference to the given PARCScheduledTask instance, decrementing the reference count for the instance.

The pointer to the instance is set to NULL as a side-effect of this function.

If the invocation causes the last reference to the instance to be released, the instance is deallocated and the instance's implementation will perform additional cleanup and release other privately held references.

[in,out]instancePtrA pointer to a pointer to the instance to release.
