PARCCertificate* parcCertificate_Acquire ( const PARCCertificate *  certificate)

Increase the number of references to a PARCCertificate instance.

Note that a new PARCCertificate is not created, only that the given PARCCertificate reference count is incremented. Discard the reference by invoking parcCertificate_Release.

Parameters
[in]certificateA pointer to the original instance.
Returns
The value of the input parameter instance.

Example:

1 {
2  PARCCertificate *x = parcCertificate_CreateFromInstance(...);
3  PARCCertificate *x2 = parcCertificate_Acquire(x);
4 
5  parcCertificate_Release(&x);
6  parcCertificate_Release(&x2);
7 }

PARCCryptoHash* parcCertificate_GetCertificateDigest ( const PARCCertificate *  certificate)

Get the SHA-256 digest of the certificate.

Equivalent of (for rsa/sha256): openssl x509 -outform DER -out test_rsa_crt.der -in test_rsa.crt openssl sha256 -out test_rsa_crt_sha256.bin -sha256 -binary < test_rsa_crt.der Which is also the same as (but not in der format) openssl x509 -in test_rsa.crt -fingerprint -sha256

Parameters
[in]certificateA pointer to a PARCCertificate instance.
Returns
NULL Memory could not be allocated.
A PARCCryptoHash instance which internally contains a hash digest of the certificate.

Example:

1 {
2  PARCX509Certificate *certificate = parcX509Certificate_CreateFromPEMFile(filename);
3  PARCCertificate *wrapper = parcCertificate_CreateFromInstance(PARCX509CertificateInterface, certificate);
4 
5  PARCCryptoHash *certificateKeyDigest = parcCertificate_GetPublicKeyDigest(wrapper);
6 }

void parcCertificate_Release ( PARCCertificate **  certificateP)

Release a previously acquired reference to the specified instance, decrementing the reference count for the instance.

The pointer to the instance is set to NULL as a side-effect of this function.

If the invocation causes the last reference to the instance to be released, the instance is deallocated and the instance's implementation will perform additional cleanup and release other privately held references.

Parameters
[in,out]certificatePA pointer to a pointer to the instance to release.

Example:

1 {
2  PARCCertificate *x = parcCertificate_Acquire(...);
3 
4  parcCertificate_Release(&x);
5 }
