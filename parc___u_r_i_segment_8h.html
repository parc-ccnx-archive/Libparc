Compares two PARCURISegment instances for order.

As strings, URI segments are compared in normal lexographical order. This is analogous to strcmp(...).

Parameters
[in]aA PARCURISegment pointer, or NULL.
[in]bA PARCURISegment pointer, or NULL.
Returns
A negative integer, zero, or a positive integer as a is less than, equal to, or greater than b, accordingly.

Example:

1 {
2  PARCURISegment *segmentA = parcURISegment_Create(6, "HelloA");

6 }

Create a PARCURISegment instance copying data from the given pointer and length.

A PARCURISegment is a tuple consisting of a pointer to arbitrary memory containing the first byte of length bytes as the value of the segment.

Since the input parameter pointer points to arbitrary memory, this function makes a private copy of the data.

bool parcURISegment_Equals ( const PARCURISegment *  x,
const PARCURISegment *  y 
)

Determine if two PARCURISegment instances are equal.

The following equivalence relations on non-null PARCURISegment instances are maintained

  • It is reflexive: for any non-null reference value x, parcURISegment_Equals(x, x) must return true.
  • It is symmetric: for any non-null reference values x and y, parcURISegment_Equals(x, y) must return true if and only if parcURISegment_Equals(y x) returns true.
  • It is transitive: for any non-null reference values x, y, and z, if parcURISegment_Equals(x, y) returns true and parcURISegment_Equals(y, z) returns true, then parcURISegment_Equals(x, z) must return true.
  • It is consistent: for any non-null reference values x and y, multiple invocations of parcURISegment_Equals(x, y) consistently return true or consistently return false.
  • For any non-null reference value x, parcURISegment_Equals(x, NULL) must return false.
Parameters
[in]xA pointer to a PARCURISegment instance.
[in]yA pointer to a PARCURISegment instance.
Returns
true PARCURISegment x and y are equal.
false PARCURISegment x and y are not equal.

Example:

1 {
2  PARCURISegment *segmentA = parcURISegment_Create(5, "Hello");
3  PARCURISegment *segmentB = parcURISegment_Create(5, "Hello");
4 
5  if (parcURISegment_Equals(segmentA, segmentB)) {
6  printf("The URI segments are equal\n");
7  } else {
8  printf("The URI segments are NOT equal\n");
9  }
10 
11  parcURISegment_Equals(&segmentA);
12  parcURISegment_Equals(&segmentB);
13 }

PARCURISegment* parcURISegment_Parse ( const char *  string,
const char **  pointer 
)

Parse a single URI segment.

The input parameter string must point to a '/' which indicates the beginning of a segment. The segment is terminated by a null byte or a '?' or '#' character and may may be zero length. The output parameter pointer will be assigned the address of the first character that is not part of the parsed segment.

The returned value must be deallocated via parcMemory_Deallocate.

Parameters
[in]stringPointer to the beginning of the segment
[in,out]pointerWill be assigned the address of the first character that is not part of the parsed segment.
Returns
An allocated PARCURISegment the must be deallocated via parcMemory_Deallocate.

Example:

1 {
2  char *pointer;
3  PARCURISegment *segment = parcURISegment_Parse("lci:/a/b/", &pointer);
4  ...
5  parcBuffer_Release(&segment);
6 }

Release a previously acquired reference to the specified instance, decrementing the reference count for the instance.

The pointer to the instance is set to NULL as a side-effect of this function.

If the invocation causes the last reference to the instance to be released, the instance is deallocated and the instance's implementation will perform additional cleanup and release other privately held references.

char* parcURISegment_ToString ( const PARCURISegment *  segment)

Produce a null-terminated string representation of the specified PARCURISegment instance.

The result must be freed by the caller via parcMemory_Deallocate.

The result is percent-encoding if necessary. A PARCURISegment may contain characters that are not permitted in the path in their native form. These characters must be 'percent encoded' using the '' followed by the hexadecimal value of the character. For consistency, percent-encoded octets in the ranges of ALPHA (%41-%5A and %61-%7A), DIGIT (%30-%39), hyphen (%2D), period (%2E), underscore (%5F), or tilde (%7E) should not be created by URI producers and, when found in a URI, should be decoded to their corresponding unreserved characters by URI normalizers.

Parameters
[in]segmentA pointer to the instance.
Returns
An allocated, null-terminated C string that must be deallocated via parcMemory_Deallocate.

Example:

1 {
2  PARCURISegment *instance = parcURISegment_Create(5, "Hello");
3 
4  char *string = parcURISegment_ToString(instance);
5 
6  printf("Hello: %s\n", string);
7  parcMemory_Deallocate((void **)&string);
8 
9  parcURISegment_Release(&instance);
10 }
