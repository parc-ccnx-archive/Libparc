PARCKeyedElement* parcKeyedElement_Create ( void *  data,
const void *  key,
size_t  keylen 
)

Create a PARCKeyedElement.

Note that the key will be copied (size keylen) while the data will just be referenced. The key copy will be released when the KeyedElement is destroyed.

Parameters
[in]dataThe data we want to associate with a key.
[in]keyA pointer to the key data (will be copied).
[in]keylenThe length of the keydata.
Returns
The new instance of PARCKeyedElement.
