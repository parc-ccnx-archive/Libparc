PARCLogReporter* parcLogReporter_Acquire ( const PARCLogReporter *  instance)

Increase the number of references to a PARCLogReporter instance.

Note that new PARCLogReporter is not created, only that the given PARCLogReporter reference count is incremented. Discard the reference by invoking parcLogReporter_Release.

Parameters
[in]instanceA pointer to a PARCLogReporter instance.
Returns
The input PARCLogReporter pointer.

Example:

1 {
2  PARCLogReporter *reporter = parcLogReporter_Create(&parcLogReporterFile_Acquire,
3  parcLogReporterFile_Release,
4  parcLogReporterFile_Report,
5  parcOutputStream_Acquire(output));
6 
7  PARCLogReporter *x_2 = parcLogReporter_Acquire(reporter);
8 
9  parcLogReporter_Release(&reporter);
10  parcLogReporter_Release(&x_2);
11 }

void parcLogReporter_Release ( PARCLogReporter **  instancePtr)

Release a previously acquired reference to the specified instance, decrementing the reference count for the instance.

The pointer to the instance is set to NULL as a side-effect of this function.

If the invocation causes the last reference to the instance to be released, the instance is deallocated and the instance's implementation will perform additional cleanup and release other privately held references.

Parameters
[in,out]instancePtrA pointer to a pointer to a PARCLogReporter. The parameter is set to zero.

Example:

1 {
2  PARCLogReporter *reporter = parcLogReporter_Create(&parcLogReporterFile_Acquire,
3  parcLogReporterFile_Release,
4  parcLogReporterFile_Report,
5  parcOutputStream_Acquire(output));
6 
7  parcLogReporter_Release(&reporter);
8 }
