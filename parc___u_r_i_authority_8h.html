Increase the number of references to a PARCURIAuthority instance.

Note that new PARCURIAuthority is not created, only that the given PARCURIAuthority reference count is incremented. Discard the reference by invoking parcURIAuthority_Release.

bool parcURIAuthority_Equals ( const PARCURIAuthority *  authA,
const PARCURIAuthority *  authB 
)

Return true if two PARCURIAuthority instances are equal.

The following equivalence relations on non-null PARCURIAuthority instances are maintained:

  • It is reflexive: for any non-null reference value x, parcURIAuthority_Equals(x, x) must return true.
  • It is symmetric: for any non-null reference values x and y, parcURIAuthority_Equals(x, y) must return true if and only if parcURIAuthority_Equals(y x) returns true.
  • It is transitive: for any non-null reference values x, y, and z, if parcURIAuthority_Equals(x, y) returns true and parcURIAuthority_Equals(y, z) returns true, then parcURIAuthority_Equals(x, z) must return true.
  • It is consistent: for any non-null reference values x and y, multiple invocations of parcURIAuthority_Equals(x, y) consistently return true or consistently return false.
  • For any non-null reference value x, parcURIAuthority_Equals(x, NULL)) must return false.
Parameters
[in]authAThe first PARCURIAuthority instance.
[in]authBThe second PARCURIAuthority instance.
Returns
true the given PARCURIAuthority instances are equal.
false the given PARCURIAuthority instances are not equal.

Example:

1 {
2  const char *uriString = "http://chris@parc.com:80";
3  PARCURI *uri = parcURI_Parse(uriString);
4  PARCURI_Authority *authority = parcURIAuthority_Parse(parcURI_GetAuthority(uri));
5  PARCURI_Authority *handle = parcURIAuthority_Acquire(authority);
6 
7  if (parcURIAuthority_Equals(authority, handle)) {
8  printf("Authorities are equal.\n");
9  } else {
10  printf("Authorities are NOT equal.\n");
11  }
12 
13  parcURIAuthority_Release(&uri);
14  parcURIAuthority_Release(&copy);
15 }

void parcURIAuthority_Release ( PARCURIAuthority **  authPtr)

Release a previously acquired reference to the specified instance, decrementing the reference count for the instance.

The pointer to the instance is set to NULL as a side-effect of this function.

If the invocation causes the last reference to the instance to be released, the instance is deallocated and the instance's implementation will perform additional cleanup and release other privately held references.

Parameters
[in,out]authPtrA pointer to a pointer to the instance to release.

Example:

1 {
2  PARCURIAuthority *auth = parcURIAuthority_Create();
3 
4  parcURIAuthority_Release(&auth);
5 }
