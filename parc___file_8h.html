char* parcFile_ToString ( const PARCFile *  file)

Produce a null-terminated string representation of the specified PARCFile instance.

The non-null result must be freed by the caller via parcMemory_Deallocate.

Parameters
[in]fileA pointer to the PARCFile instance.
Returns
NULL Cannot allocate memory.
non-NULL A pointer to an allocated, null-terminated C string that must be deallocated via parcMemory_Deallocate.

Example:

1 {
2  PARCFile *instance = parcFile_Create("/tmp/foo");
3 
4  char *string = parcFile_ToString(instance);
5 
6  if (string != NULL) {
7  printf("Hello: %s\n", string);
8  parcMemory_Deallocate((void **)&string);
9  } else {
10  printf("Cannot allocate memory\n");
11  }
12 
13  parcFile_Release(&instance);
14 }
