Memory and Buffer Management

Detailed Description

A Facade to memory allocation features.

PARC Memory provides an interface implementing many regularly available memory allocation functions. This interface is a Facade that software implementors may use to substitute different kinds of underlying Interfaces of these allocation fucntions. Notable examples are PARC Safe Memory and PARC Stdlib Memory.

int parcMemory_MemAlign ( void **  pointer,
const size_t  alignment,
const size_t  size 
)

Allocate aligned memory.

Allocates size bytes of memory such that the allocation's base address is an exact multiple of alignment, and returns the allocation in the value pointed to by pointer.

The requested alignment must be a power of 2 greater than or equal to sizeof(void *).

Memory that is allocated can be used as an argument in subsequent call Reallocate, however Reallocate is not guaranteed to preserve the original alignment.

Parameters
[out]pointerA pointer to a void * pointer that will be set to the address of the allocated memory.
[in]alignmentA power of 2 greater than or equal to sizeof(void *)
[in]sizeThe number of bytes to allocate.
Returns
0 Successful
EINVAL The alignment parameter is not a power of 2 at least as large as sizeof(void *)
ENOMEM Memory allocation error.

Example:

1 {
2  void *allocatedMemory;
3 
4  int failure = parcMemory_MemAlign(&allocatedMemory, sizeof(void *), 100);
5  if (failure == 0) {
6  parcMemory_Deallocate(&allocatedMemory);
7  // allocatedMemory is now equal to zero.
8  }
9 }

void* parcMemory_Reallocate ( void *  pointer,
size_t  newSize 
)

Try to change the size of the allocation pointed to by pointer to newSize, and returns ptr.

If there is not enough room to enlarge the memory allocation pointed to by pointer, create a new allocation, copy as much of the old data pointed to by pointer as will fit to the new allocation, deallocate the old allocation, and return a pointer to the allocated memory.

If pointer is NULL, simply invoke the parcMemory_Allocate function to allocate memory aligned to the value of sizeof(void *) of newSize bytes. If newSize is zero and pointer is not NULL, a new, minimum sized object is allocated and the original object is freed.

When extending a region previously allocated with AllocateAndClear, the additional memory is not guaranteed to be zero-filled.

Parameters
[in]pointerA pointer to previously allocated memory, or NULL.
[in]newSizeThe size of the allocated memory.
Returns
non-NULL A pointer to allocated memory.
NULL A an error occurred.

Example:

1 {
2  void *allocatedMemory;
3 
4  allocateMemory = parcMemory_Allocate(100);
5 
6  allocatedMemory = parcMemory_Reallocate(allocatedMemory, sizeof(void *), 200);
7 
8  parcMemory_Deallocate(&allocatedMemory);
9 }
